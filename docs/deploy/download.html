Android 安装步骤

  1. 点击上方「Android 下载」按钮，浏览器开始下载 APK
  2. 下载完成后从通知栏点击 APK，系统提示「禁止安装来自此来源的应用」
  3. 进入 设置 → 应用 → 特殊应用权限 → 安装未知应用，授权当前浏览器
  4. 回到下载页重新点击 APK，完成安装
  5. 首次启动 App 授予「位置」「相机」「存储」权限

iOS 安装步骤（TestFlight 灰度）

  1. App Store 搜索「TestFlight」并安装
  2. 收到邀请邮件后，点击邮件中的「View in TestFlight」链接
  3. TestFlight 内点击「Install」，App 图标出现在桌面
  4. 若超过 90 天未接受邀请，需联系管理员重新发送

常见问题

安装时提示「解析包出现问题」
通常是下载中断导致 APK 不完整，请重新下载。
启动后白屏
检查后端 API 地址。在 App 内「我的 → 关于」可查看当前 API endpoint。
GPS 一直拿不到
检查是否授予「位置」权限，并打开手机 GPS 开关。
iOS TestFlight 邀请邮件没收到
检查垃圾邮件；超过 90 天邀请失效，需联系管理员重新生成。
